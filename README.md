## Introduction

Windows 12 Web Version is an open source project created by **Mayank Mishra**, which uses standard web technologies (HTML, CSS, and JavaScript) to simulate and innovate operating systems on the web.

Beautiful UI design, smooth and rich animations, and various advanced features.

> Mobile device support is limited - use the desktop version for the best experience.

## Features

- Start Menu with pinned and recommended apps
- Multiple built-in applications (Calculator, Notepad, File Explorer, Terminal, etc.)
- Microsoft Edge browser simulation
- Settings app with personalization options
- Dark mode support
- AI Copilot integration
- Desktop widgets
- Task Manager
- Window management (resize, minimize, maximize)
- PWA support
